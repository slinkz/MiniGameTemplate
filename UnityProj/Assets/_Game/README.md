# _Game

这是你的游戏开发区域。所有游戏特定的代码、资源、SO 资产都放在这里。

## 目录结构
```
_Game/
├── Scripts/              # 游戏逻辑脚本
├── Scenes/               # 游戏场景
├── ScriptableObjects/    # 游戏专用 SO 资产
├── Art/                  # 美术资源
├── Audio/                # 音频资源
├── FairyGUI/             # FairyGUI 导出资源
└── Prefabs/              # 预制件
```

## 注意
- 不要修改 `_Framework/` 中的代码（除非你在改进模板本身）
- 通过 SO 事件和变量与框架层通信
- 遵守 `Docs/CONVENTIONS.md` 中的编码规范
