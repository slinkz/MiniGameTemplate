# ScriptableObjects 目录

存放游戏层的 ScriptableObject 资产实例。

## 目录约定
- `Events/` — GameEvent 实例
- `Variables/` — Variable 实例（Int/Float/Bool/String）
- `Audio/` — AudioClipSO / AudioLibrary 实例
- `Config/` — GameConfig / AssetConfig 等配置实例

## 创建方式
- 使用 `Tools → MiniGame Template → Create → ...` 菜单
- 或使用 `Tools → MiniGame Template → SO Creation Wizard`
- 或 Project 面板右键 → Create → MiniGameTemplate → ...
